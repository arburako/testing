Overview

Konefix is a beautiful portfolio theme based on Bootstrap 3. Ideal for creative people or agencies.

Thanks for using Konefix.

Theme Configuration

All theme css files located in /css/themes/..

If you want to change the color style, you have to add in the head section the single code like this one: <link rel="stylesheet" href="css/themes/color-2.css">

Fonts

Konefix uses the fonts Roboto, Roboto Condensed and Alegreya from Google Fonts.

@import url(http://fonts.googleapis.com/css?family=Roboto|Roboto+Condensed|Alegreya:700);
